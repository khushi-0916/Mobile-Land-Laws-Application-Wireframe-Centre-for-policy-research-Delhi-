## 🧾 About

**MILL (Mobile Indian Land Laws)** is a comprehensive mobile 
application designed to provide citizens with accessible, 
searchable access to Indian land laws across India.

This project was independently designed and strategised during 
a paid internship at the **Centre for Policy Research (CPR), 
New Delhi** — one of India's most prestigious independent 
policy research institutions.

As the **sole intern**, I was responsible for the complete 
UX design and mobile app development strategy end to end.

---

## 🎯 My Responsibilities

- Designed complete UI/UX wireframe for all 11 screens
- Built full mobile app development strategy document
- Proposed complete tech stack and 5 phase development roadmap
- Designed user flows for legal document discovery
- Focused on accessibility and readability for civic users

---

## 📱 Key Features Designed

| Feature | Details |
|---------|---------|
| 🗺️ Interactive Map | Full-screen map of India with region-based law access |
| 🔍 Smart Search | Keyword search with filters by category & date |
| 📜 Historical Context | Timelines of pre & post-colonial land laws |
| 🌐 Multi-language | English + regional Indian languages |
| 📥 Offline Access | Download laws for offline reading |
| 🌙 Dark Mode | Reading-friendly dark mode |
| 📊 Legal Summaries | Simplified summaries of complex legal documents |
| 🔔 Notifications | Updates on changes to land laws |

## 🛠️ Proposed Tech Stack

| Layer | Technology |
|-------|------------|
| Frontend | React Native + TypeScript |
| Maps | Mapbox GL Native |
| Backend | Firebase (Firestore + Auth) |
| Search | Algolia Full-Text Search |
| State | Redux + Redux Persist |
| Offline | AsyncStorage |
| Languages | i18next |
| PDF | react-native-pdf |
| Notifications | Firebase Cloud Messaging |
| Sharing | react-native-share |

---

## 🗺️ Development Phases
```
Phase 1 → Core Functionality (Auth, Map, Search)
Phase 2 → Advanced Features (Offline, Multilingual, Filters)
Phase 3 → UX Refinement & Performance
Phase 4 → Legal Research Tools
Phase 5 → Testing, Security & Deployment
```
